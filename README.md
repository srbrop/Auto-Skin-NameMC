# Auto-Skin-NameMC
Extensión de automatización en Node.js diseñada para subir secuencialmente tu lote de 27 skins a Minecraft y forzar la sincronización de tu historial en NameMC. Elimina por completo el proceso manual, completando el despliegue inverso de tu Skin Art en menos de 8 minutos de forma segura y automática.
